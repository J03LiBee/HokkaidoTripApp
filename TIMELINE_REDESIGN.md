- 符合從左到右的閱讀習慣

### 2. 時間更突出
- 獨立的時間欄位
- 使用等寬字體
- 易於快速定位時間點

### 3. 卡片空間更大
- 沒有圓點和線條佔用空間
- 卡片可以更寬
- 圖片顯示更大更清晰

### 4. 移動設備友好
- 簡潔的佈局
- 不需要精準點擊圓點
- 整個卡片都可點擊

## 🔧 技術實現

### 佈局結構

```jsx
<div className="flex gap-4 items-start">
  {/* 左邊：時間 */}
  <div className="flex-shrink-0 w-16 pt-4">
    <span className="text-sm font-bold text-slate-600 font-mono">
      {item.time}
    </span>
  </div>
  
  {/* 右邊：活動卡片 */}
  <GlassCard className="flex-1 ...">
    <h3>{item.title}</h3>
    <span>{item.location}</span>
    <span>{item.type}</span>
    {item.imageUrl && <img src={item.imageUrl} />}
  </GlassCard>
</div>
```

### CSS 設計

- **時間寬度**：`w-16`（64px，固定）
- **對齊方式**：`items-start`（頂部對齊）
- **間距**：`gap-4`（16px）
- **卡片內邊距**：`py-4 px-5`（上下 16px，左右 20px）

## 🧪 測試場景

1. 點擊任一活動卡片
2. **預期：** 
   - 彈出預覽 Modal
   - 顯示完整資訊和地圖

### 測試 3: 9天日期

1. 查看頂部日期選擇器
2. **預期：** 
   - 顯示 9 個日期按鈕
   - 從 12/31 到 1/8
   - 可以橫向滑動查看所有日期

### 測試 4: 有圖片的活動

1. 找一個有圖片的活動
2. **預期：** 
   - 圖片顯示在卡片底部
