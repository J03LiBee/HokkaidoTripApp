# 如何獲取正確的 Google Maps 連結

## ✅ 正確方式（3 種）

### 方法 1：從瀏覽器網址列複製（推薦）

1. 在電腦上打開 Google Maps
2. 搜尋地點（例如：手稻滑雪場）
3. **從瀏覽器網址列複製完整 URL**

```
✅ 正確格式：
https://www.google.com/maps/place/手稻滑雪場/@43.0642,141.3469,15z
https://www.google.com/maps/@43.064171,141.346939,17z
https://www.google.com/maps/search/札幌+拉麵
```

### 方法 2：使用嵌入地圖代碼

1. Google Maps → 點擊「分享」
2. 點擊「嵌入地圖」標籤
3. 複製 `<iframe>` HTML 代碼

```html
✅ 正確格式：
<iframe src="https://www.google.com/maps/embed?pb=..." width="600" height="450"></iframe>
```

### 方法 3：手動輸入地點名稱

直接在輸入框輸入地點名稱：

```
✅ 直接輸入：
手稻滑雪場
札幌市
北海道神宮
```

系統會自動轉換成地圖連結。

## ❌ 無法使用的格式

### 短連結（無法嵌入）

```
❌ 這些無法使用：
https://maps.app.goo.gl/xxxxx
https://goo.gl/maps/xxxxx
```

**原因：** 短連結是重定向 URL，無法直接嵌入地圖。

**解決方案：** 改用完整 URL（方法 1）

## 📱 手機 vs 電腦

### 在手機上
- ❌ 分享 → 複製連結 = 短連結（無法使用）
- ✅ 改用電腦獲取完整 URL

### 在電腦上  
- ✅ 從網址列複製 = 完整 URL（可以使用）
- ✅ 嵌入地圖 = iframe HTML（可以使用）

## 🔍 如何辨識正確格式

### ✅ 可以使用
```
包含以下任一特徵：
- /place/地點名稱/
- @緯度,經度
- /search/搜尋詞
- <iframe src="...">
- google.com/maps/embed
```

### ❌ 無法使用
```
包含以下特徵：
- maps.app.goo.gl
- goo.gl/maps
- 很短的連結（通常少於 30 字元）
```

## 🛠️ 故障排查

### 問題：地圖顯示在錯誤的位置（非洲）

**原因：** 使用了短連結或無效格式

**解決：**
1. 檢查 URL 格式
2. 確認不是 `maps.app.goo.gl` 開頭
3. 改用完整 URL

### 問題：地圖沒有顯示

**原因：** URL 格式無法識別

**解決：**
1. 使用嵌入代碼（`<iframe>`）
2. 或直接輸入地點名稱

### 問題：只能在手機上操作

**解決方案：**
1. 在手機 Google Maps 搜尋地點
2. 用電腦打開同一地點
3. 從電腦網址列複製 URL
4. 用電腦操作應用程式

或者：
1. 直接在輸入框輸入地點名稱（不需要 URL）

## 📝 快速參考

|------|------|---------|
| 電腦網址列 | 完整 URL | ✅ |
| 嵌入地圖 | iframe HTML | ✅ |
| 手機分享 | 短連結 | ❌ |
| 直接輸入 | 地點名稱 | ✅ |

## 🎯 最佳實踐

### 推薦順序

1. **最簡單**：直接輸入地點名稱
   - 例如：手稻滑雪場

2. **最準確**：從電腦網址列複製完整 URL
   - 包含精確座標

3. **最保險**：使用嵌入地圖代碼
   - Google 官方格式，100% 可用
